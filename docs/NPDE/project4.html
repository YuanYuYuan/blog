Project 4¶

Sod Shock Tube problem is test case commonly used in computational hydrodynamics (CFD, SPH, etc) to see how well a certain computational approach works.

It also helps in revealing the flaws in an approach (numerical scheme, etc).

The implementation uses the initial conditions as proposed by G.A. Sod in ”A Survey of Several Finite Difference Methods for Systems of Nonlinear Hyperbolic Conservation Laws”, JCP (1978).

Conservative Finite Difference Schemes¶

We can write the schemes for solving the conservation law equation

$$u_t + f(u)_x = 0$$

in the explicit form

$$u^{n+1}_j=u^{n}_j-\frac{\Delta t}{\Delta x}(h^{n}_{j+1/2}-h^{n}_{j-1/2})$$

where $h$ is a consistent numerical flux function of $u^{n}_j$, $u^{n}_{j+1}$, ...

$ u^{n}_j=u(j\Delta x, n\Delta t) $, $\Delta x, \Delta t$ are the partition of space and time respectively.

For the convenience, $u_j$ represnts $u$ in a certain time, so the iterative formula becomes $$u_{j+1}=u_j-\frac{\Delta t}{\Delta x}(h_{j+1/2}-h_{j-1/2})$$

In [4]:
def update(u, h):
    return u - dt/dx*(h - np.roll(h, 1, axis=1))

Gas Dynamics¶

$$u = \begin{pmatrix} \rho \\ m \\ e \end{pmatrix},\ f(u)= \begin{pmatrix} m \\ \frac{m^2}{\rho}+p \\ \frac{m(e+p)}{\rho} \end{pmatrix} $$

$$ p = (\gamma-1)(e-\frac{m^2}{2\rho}),\ \gamma = 1.4 $$

$$ v = \frac{m}{\rho}$$

where $\rho$ is the density, $m$ is the momentum, $e$ is the total energy, $p$ is the pressure, $\gamma$ is the ratio of heat capacity. and $v$ is the velocity.

* Conversion between u and state¶

gamma = 1.4

def u2state(u):
    rho = u[0]
    m = u[1]
    state = np.copy(u)
    state[1] = m / rho
    state[2] = get_p(u)
    return state


def state2u(state):
    rho = state[0]
    v = state[1]
    p = state[2]
    u = np.copy(state)
    m = v / rho
    e = p / (gamma-1) + (rho*v**2)/2
    u[1] = m
    u[2] = e
    return u

def get_p(u):
    rho = u[0]
    m = u[1]
    e = u[2]
    return (gamma-1)*(e-m**2/(2*rho))

def flux(u):
    rho = u[0]
    m = u[1]
    e = u[2]

    p = get_p(u)
    f = np.copy(u)
    f[0] = m
    f[1] = m**2/rho + p
    f[2] = m*(e+p)/rho
    return f

Scheme: Godunov¶

The numerical flux is given by

$$h_{j+1/2}=f(\frac{u_{j+1}+u_{j}}{2}-\frac{\Delta t}{\Delta x}(f_{j+1}-f_{j}))$$

$$h_{j-1/2}=f(\frac{u_{j}+u_{j-1}}{2}-\frac{\Delta t}{\Delta x}(f_{j}-f_{j-1}))$$

In [9]:
def Godunov(u):
    f = flux(u)
    mean_u = (np.roll(u, -1, axis=1)+u)/2
    delta_f = np.roll(f, -1, axis=1)-f
    num_flux = flux(mean_u - dt/dx*delta_f)
    return update(u, num_flux)

Scheme: Lax-Wendroff¶

The numerical flux is given by

$$h_{j+1/2}=f(\frac{u_{j+1}+u_{j}}{2}-\frac{\Delta t}{2 \Delta x}(f_{j+1}-f_{j}))$$

$$h_{j-1/2}=f(\frac{u_{j}+u_{j-1}}{2}-\frac{\Delta t}{2 \Delta x}(f_{j}-f_{j-1}))$$

def Lax_Wendroff(u):

    mean_u = (np.roll(u, -1, axis=1)+u)/2
    delta_f = np.roll(f, -1, axis=1)-f

Scheme: Lax-Friedrichs¶

The numerical flux is given by

$$h_{j+1/2}=\frac{1}{2}(f_{j+1}+f_{j})-\frac{\Delta x}{2 \Delta t}(u_{j+1}-u_{j})$$

$$h_{j-1/2}=\frac{1}{2}(f_{j}+f_{j-1})-\frac{\Delta x}{2 \Delta t}(u_{j}-u_{j-1})$$

In [11]:
def Lax_Friedrichs(u):
    f = flux(u)
    mean_f = (np.roll(f, -1, axis=1)+f)/2
    delta_u = np.roll(u, -1, axis=1)-u
    num_flux = mean_f - dx/dt * delta_u/2
    return update(u, num_flux)

Scheme: Runge-Kutta & WENO¶

Compute max flux speed, df¶

$$ \max(|v|+\sqrt{\frac{\gamma \rho}{p}}) $$

In [12]:
def max_df(u):
    state = u2state(u)
    rho = state[0]
    v = state[1]
    p = state[2]
    sound_speed = np.sqrt(gamma*rho/p)
    return np.max(abs(v) + sound_speed)

WENO3¶

In [13]:
def WENO3(u):

    epsilon = 1e-6
    sigma = max_df(u)

    # Lax-Friedrcihs flux splitting
    v = (flux(u)+sigma*u)/2
    w = np.roll((flux(u)-sigma*u)/2, -1, axis=1)

    # === Right Flux, v ===
    # chose positive fluxes
    vmm = np.roll(v,  2, axis=1)
    vm  = np.roll(v,  1, axis=1)
    vp  = np.roll(v, -1, axis=1)
    vpp = np.roll(v, -2, axis=1)

    # Polynomials
    p0n = (2*vmm - 7*vm + 11*v)/6
    p1n = (-vm + 5*v + 2*vp)/6
    p2n = (2*v + 5*vp - vpp)/6

    # Smooth Indicators (Beta factors)
    B0n = 13/12*(vmm-2*vm+v)**2 + 1/4*(vmm-4*vm+3*v)**2
    B1n = 13/12*(vm-2*v+vp)**2 + 1/4*(vm-vp)**2
    B2n = 13/12*(v-2*vp+vpp)**2 + 1/4*(3*v-4*vp+vpp)**2

    # Constants
    d0n = 1/10
    d1n = 6/10
    d2n = 3/10

    # Alpha weights
    alpha0n = d0n/(epsilon + B0n)**2
    alpha1n = d1n/(epsilon + B1n)**2
    alpha2n = d2n/(epsilon + B2n)**2
    alphasumn = alpha0n + alpha1n + alpha2n

    # ENO stencils weigths
    w0n = alpha0n/alphasumn
    w1n = alpha1n/alphasumn
    w2n = alpha2n/alphasumn

    # Numerical Flux at cell boundary
    hn = w0n*p0n + w1n*p1n + w2n*p2n

    # === Left Flux, w ===
    # chose negative fluxes
    wmm = np.roll(w,  2, axis=1)
    wm  = np.roll(w,  1, axis=1)
    wp  = np.roll(w, -1, axis=1)
    wpp = np.roll(w, -2, axis=1)

    # Polynomials
    p0p = (-wmm + 5*wm + 2*w)/6
    p1p = (2*wm + 5*w - wp)/6
    p2p = (11*w - 7*wp + 2*wpp)/6

    # Smooth Indicators (Beta factors)
    B0p = 13/12*(wmm-2*wm+w)**2 + 1/4*(wmm-4*wm+3*w)**2
    B1p = 13/12*(wm-2*w+wp)**2 + 1/4*(wm-wp)**2
    B2p = 13/12*(w-2*wp+wpp)**2 + 1/4*(3*w-4*wp+wpp)**2

    # Constants
    d0p = 3/10
    d1p = 6/10
    d2p = 1/10

    # Alpha weights
    alpha0p = d0p/(epsilon + B0p)**2
    alpha1p = d1p/(epsilon + B1p)**2
    alpha2p = d2p/(epsilon + B2p)**2
    alphasump = alpha0p + alpha1p + alpha2p

    # ENO stencils weigths
    w0p = alpha0p/alphasump
    w1p = alpha1p/alphasump
    w2p = alpha2p/alphasump

    # Numerical Flux at cell boundary
    hp = w0p*p0p + w1p*p1p + w2p*p2p

    # Compute finite volume residual term, df/dx.
    df = (hp-np.roll(hp, 1, axis=1)+hn-np.roll(hn, 1, axis=1))/dx
    return df

    # chose positive fluxes
    vmmm = np.roll(v,  3, axis=1)
    vmm  = np.roll(v,  2, axis=1)
    vm   = np.roll(v,  1, axis=1)
    vp   = np.roll(v, -1, axis=1)
    vpp  = np.roll(v, -2, axis=1)
    vppp = np.roll(v, -3, axis=1)

    # Polynomials

Runge Kutta method order 2¶

$$ u^* = u^n - \Delta t df(u^n), $$

$$ u^{n+1} = u^n - \Delta t \frac{df(u^n)+df(u^*)}{2} $$

where $df$ comes from WENO.

WENO3 + RK2¶

In [15]:
def WENO3_RK2(u):
    df = WENO3(u)
    u_star = u - dt*df
    u_star[:, 0] = u[:, 0]
    u_star[:, -1] = u[:, -1]
    return u - dt*(df + WENO3(u_star))/2
