    Mount Android in Linux

    less than 1 minute read

    To communicate with Android Phone in Linux, we have to use MTP, Media Transfer Protocol.

    Installation

    Linux MTP support is provided by libmtp package.

    For CLI, I recommend use AUR package simple-mtpfs.

    For GUI, I recommend use Thunar file manager, gvfs virtual filesystem implementation for GIO, gvfs-mtp MTP backend, thunar-volman Thunar volume manager.

    Usage of simple-mtpfs

    List detected devices

    $ simple-mtpfs -l
1: AsusFonepad 7 (FE375CXG)

    Mount device 1 on ~/mnt

    $ simple-mtpfs --device 1 ~/mnt

    Unmount mountpoint ~/mnt

    $ fusermount -u ~/mnt

    Configuration of Thunar

    To let Thunar handle automatice mounting, one must launch Thunar in daemon mode.

    So execute thunar --daemon as login, one may add this command into i3 config file for startup.

    And launch Thunar and go to Edit > Preferences, check Enable Volume Managerment under the Advanced tab.

    Check Mount removable drivers when hot-plugged, Mount removable media wehn inserted.
